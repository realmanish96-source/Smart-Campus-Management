# Smart Campus Management System (SCMP)

A simple, beginner-level Desktop application built using Python and Tkinter for managing basic campus activities.

## Features
- **Login System**: Secure login with hardcoded credentials (`student` / `1234`).
- **Dashboard**: Simple navigation hub to access all modules.
- **Notices**: View the latest campus announcements and updates.
- **Attendance**: Check static attendance percentages for different subjects.
- **Complaints**: Submit feedback or issues through a dedicated complaint box.

## Technologies Used
- **Python**: Core programming language.
- **Tkinter**: Standard GUI library for creating the desktop windows.

## Project Structure
```
SCMP/
│
├── main.py          # Entry point
├── login.py         # Login screen logic
├── dashboard.py     # Main menu screen
├── notices.py       # Notice board module
├── attendance.py    # Attendance display module
├── complaints.py    # Complaint submission module
├── assets/          # Logo and icon storage
└── README.md        # Project documentation
```

## How to Run
1. Make sure you have Python installed on your system.
2. Open a terminal or command prompt.
3. Navigate to the `SCMP` folder.
4. Run the following command:
   ```bash
   python main.py
   ```
